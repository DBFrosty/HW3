# HW3
Bad code for Math 4370, Fall 2025. Homework 3 asks students to turn this into better code

## Build Instructions

Paste in terminal:
```
g++ "problem.cpp" "exporter.cpp" "simple_test_generator.cpp" -o simple_test_generator.exe
```
```
g++ "problem.cpp" "exporter.cpp" "fancy_test_generator.cpp" -o fancy_test_generator.exe
```
Then run with:
```
.\simple_test_generator.exe
```
And:
```
.\fancy_test_generator.exe
```
